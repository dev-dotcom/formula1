API Documentation

Based on simple REST principles, Formula 1 Web API endpoints returns JSON metadata about Drivers and Constructors from year 2018 to 2022, directly from Formula 1 Data Catalogue.

The base address of Web API is http://127.0.0.1:5000/api. The API provides set of endpoints, each with its own unique path.

Data resources are accessed via standard HTTP requests in UTF-8 format to an API endpoint. Where possible, Web API uses GET HTTP verbs for each action.

Getting Started 🚀

This is where the magic begins! The following steps will help you to get started with your journey towards creating some awesome F1 apps using the API:

  • Register   yourself in order to start using API.
  • Once you have created registered, you will have access to the app credentials. These will be required for API authorization to obtain an access token.
  • Login with your credentials and you will get your access token.
  • Use the access token in API requests.

    • Follow this full tutorial to learn to how to make your first API call.

    How to Register

    Send POST  request to the below url:

    http://127.0.0.1:5000/api/signup

    Status code

    201 : User succesfully created.

    202 : User already exists.

    Login and generate access token 🔐

    Send POST  request to below url:

    http://127.0.0.1:5000/api/signin

    Use your registered credentials "username" and "password" as form-data KEYS and your credentials as VALUES to generate access token.

    Status code:

    201 : Logged in and token generated succesfully.

    401 : User does not exist or fill form correctly.

    403 : Wrong password

    Use token to get your account details

    Send GET request to below url:

    http://127.0.0.1:5000/api/user

    Add your generated access token to the header of this request. Use "jwt-access-token" as header KEY and "your-token" as header VALUE

    Status code:

    200 : OK

    401 : Token is missing.

    403 : Token is invalid.
