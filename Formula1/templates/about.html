About

This website has used Flask at server end.

MySQL database is used to store and manage data.

Admins can perform CRUD operations in Admin panel.

Admin panel has "session" authenticationat server end.

Users has access to table format data table format data without registration.

Users has to register him/herself to access public APIs.

50 unique API endpoints are exposed for users.

APIs are served by Flask's Jsonify.

Users authentication is managed by JWT Tokens (client-side).

Users passwords are encrypted in SHA-256 algorithm and stored in database. Provides security in case of Token Hijacking.

Admins has access to server logs for monitering and further inspections.
